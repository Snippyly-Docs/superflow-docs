## How to Comment on a File

In this article, you will learn how to comment on a file using the dashboard.

Access the Superflow Dashboard by going to [app.usesuperflow.com](app.usesuperflow.com). Alternatively, when you are in Review Mode, you can use the method below:

1. Find the **Superflow Toolbar** and click the **More Options** icon.

<Frame>
![](/images/dashboard/how-to-access-project-settings/how-to-access-project-settings/INT1FINAL.png)
</Frame>

2. A sub-menu will expand, click the **View Project** option to select.

3. A new tab will direct you to the **Project Dashboard** page.

> **Note**: By default, you will be directed to the Dashboard for Comment Status screen when accessing the **Project Dashboard**.

4. Click the **FILES** module to access the **Files** page.

<Frame>
![](/images/files/how-to-comment-on-a-file/CF1FINAL.png)
</Frame>

5. Find the file and click its **Review Link** icon.

<Frame>
![](/images/files/how-to-comment-on-a-file/CF2FINAL.png)
</Frame>

6. On the **Superflow Toolbar**, click the **Comments** icon to enable the **Comment Mode**.

<Frame>
![](/images/files/how-to-comment-on-a-file/CF3FINAL.png)
</Frame>

7. Point and click the area on the file that you want to comment on.

8. In the **Comment** field, enter your comment and click the **Send** icon.

<Frame>
![](/images/files/how-to-comment-on-a-file/CF4FINAL.png)
</Frame>
