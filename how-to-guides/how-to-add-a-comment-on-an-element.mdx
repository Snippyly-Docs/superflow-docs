## How to Add a Comment on an Element

In this article, you will learn how to add comments on an element using the Superflow Toolbar.

1. Find the **Superflow Toolbar** and click the **Comment** icon.

<Frame>
![](images/how-to-guides/how-to-add-a-comment-on-an-element/ACE1FINAL.png)
</Frame>

2. After clicking the icon, click the **Element** you want to comment on.
> **Note**: when you hover your cursor on an element, it will appear as a **Comment** icon which signifies that you selected the comment feature.

<Frame>
![](images/how-to-guides/how-to-add-a-comment-on-an-element/ACE2FINAL.png)
</Frame>

3. The **Comment Box** will appear, enter your comment or type **@** to add other users.
> **Note**: the **Comment Box** lets you add users, attach files, record audio, record videos, and record screens.

<Frame>
![](images/how-to-guides/how-to-add-a-comment-on-an-element/ACE3FINAL.png)
</Frame>

4. Click the **Post** button to post your comment.


5. Once you click the Post button, a comment thread will appear on the selected element.

<Frame>
![](images/how-to-guides/how-to-add-a-comment-on-an-element/ACE4FINAL.png)
</Frame>
