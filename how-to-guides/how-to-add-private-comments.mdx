## How to Add Private Comments

In this article, you will learn how to add private comments. This type of comment is not accessible to all users and requires admin credentials to read or respond.

1. Ensure that you are logged in to your **Superflow** account.

2. Find the **Superflow Toolbar** and click the **Private Mode** lock icon to enable this feature.

<Frame>
![](/images/how-to-guides/how-to-add-private-comments/PM1FINAL.png)
</Frame>

3. Enabling this mode will allow only the admin users to view your private comments. On the **Superflow Toolbar**, click the **Comment** icon.

<Frame>
![](/images/how-to-guides/how-to-add-private-comments/PM2FINAL.png)
</Frame>

4. Click the area or an element where you want to comment privately.

<Frame>
![](/images/how-to-guides/how-to-add-private-comments/PM3FINAL.png)
</Frame>

5. A **Comment Box** will appear. Enter the private comment and click the **Post** icon.

<Frame>
![](/images/how-to-guides/how-to-add-private-comments/PM4FINAL.png)
</Frame>
