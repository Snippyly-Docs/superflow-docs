## How to Exit the Review Mode

In this article, you will learn how to exit from the Review Mode.

1. Find the **Superflow Toolbar** and click the **More Options** icon.

<Frame>
![](/images/how-to-guides/how-to-exit-review-the-mode/ERM1FINAL.png)
</Frame>

2. A sub-menu will expand, select **Exit Review Mode**.

3. After selecting the **Exit Review Mode** option, you will immediately exit the review mode. The **Superflow Toolbar** will no longer be available.
